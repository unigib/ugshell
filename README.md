# UGShell: A C# Telnet-Style Teaching Tool

**UGShell** is an educational project designed for undergraduate students to explore the fundamentals of network programming. It provides a simplified, hands-on implementation of the **Telnet pattern**, demonstrating how a client and server communicate over the internet using C#.

## 🎯 Purpose
The primary goal of this repository is to illustrate:
* **TCP/IP Socket Programming:** Understanding how data is sent and received over a reliable stream.
* **Client-Server Architecture:** Managing connections, listening for requests, and handling responses.
* **Network Protocol Basics:** Implementing a command-line interface (shell) over a network connection.

## 🛠 Features
* **C# Implementation:** Written in clean, readable C# for easy comprehension.
* **Dual-Component Design:** Includes both a **Server** (listener) and a **Client** (requester).
* **Interactive Shell:** Allows remote command execution or message passing based on the Telnet model.
